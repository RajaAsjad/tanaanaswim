# TANA ANA Swimwear

Premium women's and family swimwear ecommerce storefront built with Next.js. A conversion-focused, mobile-first redesign inspired by the [TANA ANA](https://tanaanaswim.com/) brand.

## Features

- **Premium homepage** — Hero, category grid, brand story, reviews, water safety blog
- **Collection pages** — Women's, Children's, Accessories with filtering
- **Long Torso / Regular Torso filters** — Size-fit differentiation for women's collection
- **Product pages** — Gallery, variant selectors, trust signals, add-to-cart
- **Educational pages** — About, Fit Guide, Water Safety blog
- **Cart drawer** — Persistent cart with free shipping threshold ($125)
- **Mobile-first UX** — Responsive navigation, touch-friendly interactions
- **CRO elements** — Reviews, trust badges, newsletter signup, urgency messaging

## Tech Stack

- Next.js 16 (App Router)
- React 19
- TypeScript
- Tailwind CSS 4
- Lucide React icons

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```
